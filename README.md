# 📚 [한 입 크기로 잘라먹는 Next.js](https://www.inflearn.com/course/%ED%95%9C%EC%9E%85-%ED%81%AC%EA%B8%B0-nextjs/dashboard) 프로젝트

한 입 크기로 잘라먹는 Next.js를 수강하면서 실습한 Next.js 프로젝트입니다.

학습 기간: 2025.12 -

## ⚙️ 실행 방법

```bash
# 1. 저장소 클론
git clone

# 2. 프로젝트 폴더로 이동
cd [프로젝트 폴더명]

# 3. 의존성 설치
npm install

# 4. 개발 서버 실행
npm run dev
```
